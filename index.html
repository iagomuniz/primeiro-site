CSS: cascatas de folhas de estilo

muitos consideram ela como uma linguagem bloated, mas ela faz o que é necessário. ela tem até muito potencial. esse site usa talvez só 0.00001% do poder do CSS real. pois temos funções de perspectiva, funções lógicas, variáveis e tudo isso sem precisar recorrer a frameworks e bibliotecas... Na maioria das vezes as pessoas só escolhem uma biblioteca como Tailwind ou bootstrap e fazem a festa com ela.

Eu não posso julgá-los, ainda mais por conta de muitas bibliotecas CSS permitirem fazer o "básico bem feito" com algum nível de maestria sem escrever nenhuma linha de CSS, mas é de se notar que ele faz basicamente isso: o Básico Bem Feito. Não é nada que um cara usando um wordpress ou elementor não consigam fazer...
