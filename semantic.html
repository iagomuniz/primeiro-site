Por quê?

tags semânticas são deveras interessantes, elas podem tornar o HTML um pouco mais colorido e menos classudo. quer dizer, em lingugens de marcação são interessantes você colocar tags que dizem explicitamente o que você quer com elas. HTML tem as tags reservadas, mas como sites possuem tags tipo... seção, cabeçalho, rodapé e coisas assim, nada mais justo do que fazer o HTML ter tags para essas seções. Elas obviamente não tem nada de especial. são elementos do tipo bloco que possuem comportamento parecido com uma div. De brinde. temos o nosso SEO ficando feliz.

Uma vez o Google disse:

Prometermos que nós não usaremos mais tags em nossa engine de pesquisa. usaremos coisas como: título, descrição, o quanto você nos paga e se você utiliza tags semânticas corretamente. [...]

e após isso, a web mudou.
